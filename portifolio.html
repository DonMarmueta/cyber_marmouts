Automation of Domain and Subdomain Reconnaissance in Bash

Desenvolvimento de uma ferramenta em Bash para coleta de informações essenciais para análise de segurança, incluindo enumeração de subdomínios, escaneamento de portas e identificação de tecnologias. Utiliza Assetfinder, Nmap, WhatWeb e Gobuster, organizando os resultados para investigações detalhadas.

Competências: Bash, Penetration Testing, Cibersegurança.

ACIDD V - Ferramenta de Busca de Dados Vazados

Ferramenta de uso individual para reunir dados vazados de fontes online, utilizando conhecimentos de OSINT com Python. Ajuda na segurança digital pessoal e prevenção de fraudes.

Competências: Python, Segurança da Informação, Linux, Prevenção de Fraudes.

MVP - Security Hub and Useful Links

Plataforma centralizada de recursos para segurança cibernética, relatórios bancários, controle parental, OSINT e CTFs. Design responsivo e acessível, com ferramentas como InVID e Etherscan para análises avançadas.

Competências: HTML, UX Design, OSINT.

Automação de Análises Metagenômicas

Uso do QIIME 2 e ferramentas Linux para processar grandes volumes de dados de DNA/RNA ambiental. Inclui desenvolvimento de pipelines automatizados e conteúdos educacionais no YouTube.

Competências: Bioinformática, QIIME 2, Linux, Python.

Projeto de Mineração de Chia Coin (XCH)

Implementação de ambiente para mineração de Chia Coin, com otimização de hardware e armazenamento de 80 TB. Projeto explorou a viabilidade do modelo de mineração baseado em espaço e tempo.

Competências: Blockchain, Criptomoedas, Infraestrutura de TI.

Tecnova - Soluções Ambientais

Desenvolvimento de equipamento para controle microbiológico em lubrificantes industriais. Projeto avançado com validação comercial, resultando em aumento da vida útil dos lubrificantes.
